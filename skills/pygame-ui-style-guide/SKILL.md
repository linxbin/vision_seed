# 目标

让 VisionSeed 页面统一、清晰、儿童友好，并兼容中英文显示。

# 必守规则

- 尊重当前项目窗口和布局约束，不硬编码新的固定分辨率
- 中文和英文都必须可显示，避免方框
- 所有菜单页、帮助页、设置页都应有明确返回入口
- 倒计时、积分、模式、按钮不能互相遮挡
- 红蓝滤镜只作用于图案层，不影响背景

# 默认视觉方向

- 清晰可读
- 焦点明确
- 反馈温和
- 文案简短
- 配色稳定

# 默认交互规则

- `Enter`：确认或开始
- `Esc`：返回、取消或退出当前页
- 键盘和鼠标都应可用
- 帮助说明内容较多时优先独立页面，不塞进复杂弹层

# 默认输出

1. 先修布局和返回链路
2. 再修文案可读性和焦点状态
3. 同时检查 `zh-CN` 与 `en-US`
4. 补对应 UI 回归测试
