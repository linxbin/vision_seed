# Branding Assets

Current local branding drafts for the software:

- `shiya_mark.svg`: transparent brand mark, suitable for splash/logo usage
- `shiya_app_icon.svg`: rounded square app icon, suitable for desktop shortcuts and packaging

Design direction:

- brand name: `视芽`
- keyword blend: `eye + sprout + growth`
- visual tone: child-friendly, bright, medical-adjacent but not clinical
- palette: sky blue, teal, warm orange

Current integration status:

- executable icon wired in `visionseed.spec`
- window title can use the branded app name from `config/game.py`

If Windows packaging needs a real executable icon, export:

- `shiya_app_icon_256.png`
- `shiya_app_icon.ico`
