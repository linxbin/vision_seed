# 双眼找图案（Eye Find Patterns） 核心设计文档

# 游戏分类：同时视训练

# 一、核心定位

面向3-12岁儿童的同时视训练小游戏，5分钟/局，裸眼/红蓝眼镜双模式可选，仅保留单局积分（局结束清零），操作简单、护眼优先。

# 二、核心玩法与规则

## （一）模式选择

- 裸眼模式（默认）：左右两侧显示相同卡通图案（星星、猫咪、蝴蝶等）。

- 红蓝眼镜模式（强化）：提供左红右蓝、左蓝右红两个滤镜选项，图案按选项添加对应滤镜，需佩戴红蓝眼镜游玩。

## （二）核心操作

双眼同时注视屏幕，确认左右两个图案均清晰可见后点击屏幕；成功后随机切换新图案继续，失败仅提示重试（不扣积分、不换图案），直至5分钟结束。

## （三）关键规则

- 积分：单次成功+10积分，连续3次成功额外+20积分，单局无上限，局结束自动清零。

- 时间：单局固定5分钟，剩余30秒时倒计时变红提醒。

- 护眼：背景为柔和浅色系（无闪烁）；红蓝滤镜颜色柔和、不刺眼。

- 退出：游戏中点击右上角“小房子”可直接退出；局结束在结算页选择“继续/退出”。

# 三、红蓝眼镜模式 滤镜参数（核心）

|滤镜颜色|RGB值|透明度|饱和度|亮度|
|---|---|---|---|---|
|红色|(255,102,102)|80%|50%|70%|
|蓝色|(102,102,255)|80%|50%|70%|
备注：滤镜仅作用于图案，不影响背景；按所选选项为左右图案分别应用对应滤镜。

# 四、界面核心设计（无多余元素）

## 1. 首页

标题：双眼找图案；核心按钮：【裸眼训练】（直接进入）、【红蓝眼镜训练】（弹出滤镜选项，选后点【开始训练】）；辅助：【帮助指南】、背景：浅蓝渐变+飘动云朵/星星。

## 2. 游戏主界面

顶部：当前模式、积分、剩余时间；红蓝模式额外显示“佩戴眼镜”+“当前滤镜”提示；中间：核心游戏区（左右对应图案）；底部：滚动距离提示、单次30秒倒计时；右上角：“退出”（小房子图标）。

## 3. 结算页面

核心展示：本次时长、成功次数、获得积分、训练模式、滤镜方向（红蓝模式）；提示：“积分已清零”+护眼口诀；按钮：【继续游戏】（返回首页选模式）、【退出游戏】（返回桌面）。

# 五、核心文案（儿童向，简洁）

- 操作引导：“让两个图案都变清晰，点击屏幕吧！”“红蓝模式请佩戴眼镜哦”

- 成功提示：“太棒啦！切换新图案～”“+10积分！”

- 失败提示：“再试试，让两个图案都清晰～”

- 结算提示：“训练时间到啦！”“红蓝训练更高效哦”
